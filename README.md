**Calorie Counter**
A web-based calorie tracking application that helps users monitor their daily calorie intake and expenditure.

Features
Daily Calorie Budget: Set your daily calorie target

Meal Tracking: Log calories for Breakfast, Lunch, Dinner, and Snacks

Exercise Tracking: Record calories burned through exercise

Dynamic Entries: Add multiple food and exercise entries for each category

Real-time Calculation: Calculate remaining calories based on intake and expenditure

Responsive Design: Clean and organized layout for easy use
